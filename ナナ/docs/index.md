## 制作方針

この物語は、ナナの半生と、ナナに救われたリク・ユアが十年かけてナナを救い返す物語を両立させる。

- 本編では「救われた子供たちが、救ってくれた大人の人生を救い返す」を核にする
- ナナは強さで英雄になったのではなく、目の前の命に本気で向き合ったことで英雄になる
- リクは守護と継承の象徴にする
- ユアは救済の実行者にする
- 10年旅は単なる修行ではなく、ナナから受け取った居場所を他の孤児へ返していく旅にする
- BGM化では、1曲ごとに物語の節目と感情の核を持たせる

## 基本構成

物語は大きく三部で見る。

- ナナ編: 英雄に憧れた少女が冒険者を諦め、ギルド職員として目の前の命を救う
- リク・ユア十年旅編: ナナを救うために成長しながら、同じように孤児を助けていく
- 帰還編: 石化から目覚めたナナが、自分の人生が誰かを救っていたことを知る
