# 10年旅と地理

## 10年旅の意味

リクとユアの十年旅は、ナナを救うための修行であると同時に、ナナから受け取ったものを他の子供たちへ返していく旅でもある。

ナナはリクとユアにパンと居場所を差し出した。リクとユアは旅の中で孤児を保護し、傷ついた人を助け、やがて自分たちの家を一時保護の場所にする。

## 地理の基本

- 王都は国の中心にあるLv5都市。
- Lv4大都市は全国に6つ程度あり、孤児院の中心になる。
- ナナの街はLv3地方中心街。
- ナナの故郷は地図に載らないLv0の村。
- ナナが冒険者だった街はLv2の通り道の街。
- 西方都市とナナの街の間には山脈があり、ナナが冒険者だった街を経由しないと西方都市へ行きにくい。

このため、西方都市へ向かう用事の途中で、ナナの故郷に寄る流れが自然になる。

## 隣国の鉱山街と王女

隣国の鉱山街は、国内の鉱山都市とは別物。国内鉱山都市の山の向こう側にある隣国側の街として扱う。

2年目、リクはこの街で初の単独偵察を成功させる。その依頼で救った姫が、後に王女となる。

隣国王族には代々、石化解除魔法に適性を持つ者が生まれやすい。姫はその中でも強力な解除適性を持つ可能性が高く、王位候補である異母兄弟たちから警戒されていた。

8年目、王女が石化する。カイトたちは石化事件の原因となったモンスターの討伐へ向かい、リクとユアは友達を救うため隣国へ向かう。二人はすでに正式なSランクパーティであり、隣国側にとっても渡りに船の存在になる。

## リク・ユアの家

リク・ユアの家は孤児院ではない。二人が個人として建てた家であり、ナナと住むための家であり、一時保護の家でもある。

- ティナを含む街の孤児たちが集まる。
- 街以外から来た孤児も一時的に保護される。
- 世話になったホームレスの老人たちの家も合わせて段取りされる。
- ドラゴン襲来で設備が壊れ引退していたパン屋にも、復帰をお願いする。

資金面は安定している。理由は、王族や貴族がユアの希少性と英雄の弟子たちとの関係を重く見て、支援金と許可を出しているため。
