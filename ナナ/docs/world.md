# 世界設定

## 基本世界

- 剣と魔法の異世界。
- 複数の国が存在する。
- 成人は12歳。
- 勇者・魔王・魔族はいない。
- 英雄と呼ばれる存在は複数いる。
- ドラゴンや魔獣は存在する。

## 死亡・石化・救命

この世界では、死亡確定者と瀕死者の扱いが制度化されている。

- 死後回復はできない。
- 瀕死者は息が止まる前に石化できれば、時間停止に近い形で延命できる。
- 貴族、重要人物、大金を払える者は延命治療の対象になる。
- 判定不能、身元不明、重要人物か不明の場合は石化で時間を稼ぐ。
- 一般人と判明し、親族認定もなければ、石化体は砕かれて土へ返される。
- ドラゴンブレスを受けた場合、凍結保存は効かず、石化が唯一の選択肢になる。

## 石化解除

石化解除は極めて難しい。

- 石化解除魔法使い自体が少ない。
- 通常は石化解除と回復を別々に行うため、タイミングのズレで失敗しやすい。
- 解除と回復を一人で同時に扱える者は国に数人級。
- ユアは回復と石化解除の両方に適性があり、将来的に同時施術者として認められる。

## ドラゴン

ドラゴンは最強級の脅威。

- Aランク相当の実力者が最低3パーティ、普通なら5パーティで挑む相手。
- それでも死者が出ることがある。
- 街への襲撃は100年に一度あるかどうか。
- 強者の気配に反応して攻撃する。

ナナの街が襲われた際、ドラゴンの直線上には英雄の弟子パーティがいた。ナナたちはその進路上にいただけで、本来狙われた存在ではない。

## 孤児保護網

英雄の弟子パーティは修行がてら各地を回り、行く宛のない孤児を保護している。

- Lv4大都市の多くに、英雄の弟子たちが所有する孤児院がある。
- Lv3以下の街や村を巡回し、孤児を一時保護して最寄りのLv4孤児院へ送る。
- Lv3には最低限の保管室や保護部屋がある。
- リク・ユアの家は孤児院ではなく、個人の家兼一時保護の家。
- ナナの街周辺には近いLv4都市が偶然なく、リク・ユアの家が保護網の穴を埋める形になる。

王族や貴族は、ユアの希少性や英雄の弟子パーティとの関係を重く見て、多額の支援金や許可を出している。
