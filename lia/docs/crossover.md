# クロスオーバー設定

リアの物語と、ナナの物語を接続するための設定。

## ナナの物語との時間軸

リアが火魔法を本格的に教わる時期は、ナナが石化したばかりの頃。

この時点で、リクは5歳、ユアは4歳。

## ヴェラ

リアの火魔法の師匠は、ナナの物語に登場するヴェラ。

ヴェラは、セラとミリムの師匠でもある。

そのため、リアはセラ、ミリムと姉妹弟子の関係になる。

## 澪・陽奈の家族とのつながり

澪と陽奈は、リアを姉妹だと言っている。

澪と陽奈の両親もそれを公認している。

そのため、澪・陽奈の家族とリアは、家族ぐるみの付き合いになる。

## 李庵とユア

李庵は、将来ユアと結婚する設定。

つながり方は、リアがヴェラの弟子であること、リアと澪・陽奈の家族が家族ぐるみの付き合いをしていることから、李庵とユアがつながっていく流れ。

結婚に至る具体的なエピソードは未定。
