# タイムスケジュール

リアの冒険開始から、氷の魔女との決戦へ向かうまでの時間調整。

## 前提

- 国立魔法学校は12歳入学、3年制。
- 願書・試験の準備は入学前年の春から始まる。
- 澪と陽奈は、リアと出会った春の時点で11歳。
- 澪はもともと国立魔法学校を目指していた。
- 陽奈は家から出たくなく、近くの魔法学校に行こうかと考えていた。
- リアとの出会いと祖母の件をきっかけに、陽奈も澪と一緒に国立魔法学校を目指す。
- 澪と陽奈は翌春、12歳で国立魔法学校へ入学する。

| 時期 | 経過 | リア | 姉 |

| 2年目 冬 | 20ヶ月 | 火魔法の使い方を少し教わり、大きな火柱を立てる。火魔法の使い手だと判明する | リアが氷の魔女に対抗できる存在だと知る |
| 2年目 冬 | 20ヶ月 | 火柱を見たヴェラが現れる。リア、ルナ、村の言い伝えの話から、ヴェラはばぁばと氷の魔女が同一人物だろうと察する | ヴェラの危険度を知っているルナが、リアを守るため前に出る |
| 2年目 冬〜3年目 春 | 20〜24ヶ月 | ヴェラに弟子入りし、火魔法の修行へ入る | リアの修行を見守る |

| 4年目 春 | 36ヶ月以降 | リアが単身、氷の魔女の城へ向かう | 後にリアが単身で向かったことを知り、氷の魔女の居場所へ向かう |

## 李庵の時間軸

| 時期 | 出来事 |
|---|---|
| 1年目 春 | 澪と陽奈が仲直りし、家族の関係が修復される |
| 1年目 春・その夜 | 裏設定として、両親が安心し、夫婦として心を通わせる。その結果、後に李庵が宿る |
| 1年目 春・数日後 | 表向きには、リアが出発前に母親のお腹へ手を当て、新しい魂が宿っていることに気づく |
| 1年目 冬〜2年目 春 | 男の子が生まれる |
| 2年目 初夏〜夏 | リアが王都で澪・陽奈と再会し、弟が生まれたこと、リアから一音もらって李庵（リアン）と名付けたことを聞く |

## 表向き設定

李庵は、澪と陽奈の弟。

名前はリアから一音もらっている。

意味は、絆、人と人とのつながり。

澪と陽奈の家族にとって、リアは姉妹をつなぎ直し、家族をもう一度前へ進ませた存在。両親はその感謝を込めて、後に生まれた男の子に李庵と名付ける。

## 裏設定

澪と陽奈が仲直りしたことで、両親は長く続いていた家族の不安から解放される。

その日の晩、両親が夫婦として心を通わせたことがきっかけで、李庵が宿る。

この因果は制作側の裏設定であり、本編では直接説明しない。

表では、リアが別れ際に新しい魂を感じ取ること、後に生まれた子が李庵と名付けられたことだけを描く。
